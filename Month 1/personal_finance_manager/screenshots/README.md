# Screenshots

This folder contains screenshots of the Personal Finance Manager application.

## How to take screenshots

1. Run the app: `python main.py`
2. Navigate each menu option
3. Take a screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Save as PNG in this folder

## Recommended screenshots to capture:
- Main menu
- Add expense flow
- Category report
- Monthly report
- Search results
- Backup confirmation
